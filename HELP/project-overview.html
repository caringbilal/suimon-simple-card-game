Suimon Card Game - Project Overview

Project Structure Overview

This is a React-based card game using TypeScript, featuring turn-based gameplay with drag-and-drop mechanics. The game implements player vs AI battles with custom card artwork and animations.

src/App.tsx

Main application component managing game state and layout

  • Manages global game state using React useState
  • Implements game logic for card battles and turn management
  • Handles player and AI actions
  • Controls game flow and win/lose conditions

Key Features:

  • Player/Opponent health management
  • Card drawing and hand management
  • Battle phase implementation
  • Game state transitions

src/components/Card.tsx

Interactive card component with animations and drag-drop functionality

Component Features:

  • Drag and drop using react-dnd
  • Card stat display (Attack, Defense, HP)
  • Animation states for attacks and effects
  • Dynamic health bar with color changes

src/styles/card.css

Comprehensive styling for cards including animations and effects

Style Features:

  • Glassmorphism effects using backdrop-filter
  • Smooth animations for card interactions
  • Responsive layout adjustments
  • Dynamic health bar transitions

src/styles/player.css

Player interface and game board styling

Layout Features:

  • Health summary boxes with profile pictures
  • Card hand positioning and animations
  • Battlefield area styling
  • Player/Opponent distinct visual identities

src/types/game.ts

TypeScript type definitions for game entities


interface CardType {
    id: string;
    name: string;
    attack: number;
    defense: number;
    hp: number;
    imageUrl: string;
}

interface Player {
    id: string;
    hp: number;
    deck: CardType[];
    hand: CardType[];
}

interface GameState {
    players: {
        [key: string]: Player;
    };
    battlefield: {
        player: CardType[];
        opponent: CardType[];
    };
    currentTurn: 'player' | 'opponent';
    gameStatus: 'waiting' | 'playing' | 'finished';
}

src/utils/gameUtils.ts

Game logic and utility functions

Key Functions:

  • Initial game state setup
  • Card generation and deck management
  • Combat damage calculations
  • AI opponent logic

Getting Started

To run the project:


npm install
npm start

The game will run on localhost:3000
